# Описание

*"Это приложение с графическим интерфейсом (Tkinter) для генерации надёжных случайных паролей.  
Пользователь может задавать длину пароля, выбирать типы символов (буквы, цифры, спецсимволы) и сохранять сгенерированные пароли в историю. История автоматически загружается из JSON-файла при запуске и сохраняется при каждом добавлении нового пароля."*

# Возможности

- Ползунок для выбора длины пароля (от 4 до 50 символов)
- Чекбоксы выбора набора символов
- Генерация пароля с обязательным включением каждого выбранного типа символов
- Таблица истории с колонками: пароль, дата/время, длина, использованные наборы
- Сохранение истории в `history.json` и загрузка при старте
- Кнопка очистки всей истории
- Проверки корректности ввода (длина и выбор хотя бы одного набора)

# Требования

- Python 3.6 или новее
- Стандартные библиотеки: `tkinter`, `random`, `string`, `json`, `os`, `datetime`
